
# qemu i386/pc board bsp

This is for [Linux Lab](https://tinylab.org/linux-lab).

## Boot it

    $ ./boot.sh

## Use it in Linux Lab

    $ cd /path/to/linux-lab
    $ make B=i386/pc
    $ make boot

## Create iso and boot with qemu

Please refer to [I386 PC ISO](https://gitee.com/tinylab/i386-pc-iso).
